# Car Price Prediction (v2)

This project predicts used car prices with a more robust scikit-learn pipeline than the initial notebook-only version.

## Implemented Improvements

- `Pipeline` + `ColumnTransformer` to prevent data leakage
- `OneHotEncoder(handle_unknown="ignore")` for categorical features
- Multi-model comparison:
  - Linear Regression
  - Random Forest Regressor
  - Gradient Boosting Regressor
- 5-fold cross-validation with:
  - R2
  - MAE
  - RMSE
- Holdout test evaluation
- Best model export with `joblib`

## Project Structure

- `src/train.py`: training, model comparison, evaluation, and export
- `src/predict.py`: prediction example using the saved model
- `models/car_price_pipeline.joblib`: final trained pipeline (generated)
- `models/metrics.csv`: CV + holdout metrics (generated by script)
- `carPricePrediction.ipynb`: original notebook + v2 comparison section

## Installation

```bash
python -m pip install -r requirements.txt
```

## Train the Model

```bash
python src/train.py
```

## Run a Sample Prediction

```bash
python src/predict.py
```

## Notes

- The target `Price` is modeled as `log(Price)` and converted back with `exp` for real-price metrics/predictions.
- The training script removes missing values and filters invalid records (`Price <= 0`, `EngineV <= 0`, `Year < 1950`).
